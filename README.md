# Ad Gallery

Simple Next.js ad creative gallery with minimal UI.

## Features

- Photo/Video toggle filter
- Grid layout of ad cards
- View details page with media display
- Related ads carousel

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
